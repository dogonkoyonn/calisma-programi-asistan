# StudyPlan - Akilli Calisma Asistani

Kisisellestirilmis calisma programlari olustur, kaynak kesfet, ilerlemeni takip et!

## Hizli Baslangic

```bash
# Projeyi ac
index.html dosyasini tarayicida ac

# VEYA Live Server kullan (VSCode)
```

## Ozellikler

- Program olusturma sihirbazi
- Pomodoro zamanlayici
- Gunluk log ve ilerleme takibi
- Rozet sistemi
- Dark/Light tema
- PDF export
- PWA destegi (offline calisir)
